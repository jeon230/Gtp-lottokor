# 🎯 GTP-KOR Lotto Recommendation System

본 저장소는 대한민국 로또 6/45의 번호 추천 시스템을 GPU 기반으로 구현한 프로젝트입니다.

## ✅ 주요 기능
- 최신 로또 회차 데이터를 기반으로 한 통계 분석
- CuPy 기반의 GPU 연산으로 고속 필터링
- 통계 기반 점수화 + 추천 번호 추출
- HTML/PDF 시각화 포함

## 📂 디렉토리 구성
- `src/`: 메인 소스코드
- `data/`: 엑셀 및 설정 데이터
- `results/`: 추천 결과 저장 폴더 (업로드 제외)

## 🚀 실행 방법
```bash
cd src/
python result-gpu.py
```

## 🧠 조건 정리
- 최근 15회 미출현 번호 보너스
- 최근 20회 출현 빈도 점수
- 총합 범위 90~180
- 과거 당첨 번호와의 중복 제외
